# Budgez - Quotazioni che hanno senso. Con l'AI.

Budgez è un'applicazione web avanzata per la creazione, gestione e analisi di preventivi, che combina strumenti di calcolo rigorosi con l'intelligenza artificiale per rendere il processo di preventivazione più rapido, preciso ed efficace.

## Panoramica del Progetto

Budgez permette agli utenti di:

- **Calcolare preventivi in modo rigoroso e preciso**: Definire attività, aggiungere risorse necessarie, applicare margini commerciali e sconti.
- **Generare budget con l'AI**: Descrivere un progetto all'intelligenza artificiale e ottenere una struttura di budget completa in pochi secondi.
- **Gestire e condividere preventivi**: Organizzare, archiviare e condividere preventivi con clienti e collaboratori.
- **Monitorare lo stato dei preventivi**: Tenere traccia dell'apertura e dell'accettazione dei preventivi inviati.
- **Analizzare le performance**: Affinare progressivamente la capacità di previsione dei costi attraverso l'analisi dello storico.

## Tecnologie Utilizzate

- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: Supabase per autenticazione e database
- **Intelligenza Artificiale**: Integrazione con Claude AI per la generazione assistita dei budget
- **UI Components**: Radix UI, shadcn/ui
- **Email**: React Email, Resend per la comunicazione con i clienti
- **Grafici e Visualizzazioni**: Recharts per visualizzare dati e analisi

## Struttura dell'Applicazione

- **Dashboard**: Gestione di tutti i preventivi creati e ricevuti
- **Editor di Budget**: Interfaccia completa per la creazione e modifica dei preventivi
- **Generazione AI**: Assistente per la creazione rapida di preventivi basati su descrizioni testuali
- **Condivisione**: Sistema per inviare preventivi ai clienti con monitoraggio delle interazioni
- **Impostazioni**: Gestione del profilo utente e delle preferenze

## Funzionalità Chiave

- **Calcolo Preventivi**: Strumenti per il calcolo preciso dei costi delle attività
- **Templates**: Utilizzo di modelli predefiniti per accelerare la creazione di preventivi
- **AI Budget**: Generazione di strutture di budget complete tramite intelligenza artificiale
- **Collaborazione**: Possibilità di collaborare sui preventivi con diversi livelli di accesso (owner, editor, viewer)
- **Monitoraggio**: Tracciamento dell'interazione dei clienti con i preventivi inviati
- **Analisi**: Strumenti per analizzare l'accuratezza delle stime e migliorare i preventivi futuri

## Iniziare

```bash
# Installare le dipendenze
npm install

# Avviare il server di sviluppo
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) con il tuo browser per vedere il risultato.
